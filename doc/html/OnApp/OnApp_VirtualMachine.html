Virtual Machines

The Virtual Machine class represents the Virtual Machines of the OnAPP installation.

The ONAPP_VirtualMachine class uses the following basic methods: load, save, delete, and getList.

For full fields reference and curl request details visit: ( http://onapp.com/docs/ )
